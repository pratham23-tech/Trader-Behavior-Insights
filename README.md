# Trader Behavior Insights under Market Sentiment

## Objective
Analyze how market sentiment (Fear & Greed Index) impacts trader behavior and performance
using historical trading data.

## Dataset
- Bitcoin Fear & Greed Index
- Historical Trader Execution Data (Hyperliquid)

> Raw datasets are not included due to size/privacy.

## Key Insights
- Fear regimes show higher average trader profitability
- Extreme Greed favors SELL-side performance
- Trader behavior shifts significantly with sentiment
- Certain traders perform consistently better under specific regimes

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook
